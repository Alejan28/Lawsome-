## Project theme:
Create a synthetically-generated dataset of juridical documents, using the approach from [TF1-EN-3M](https://arxiv.org/abs/2504.20605) . 

Categories of documents:
1. Contracts (contracte) - generated by Muntean Larisa
2. Requests (cereri) - generated by Salejan Maria
3. Power of Attorney (procuri) - generated by Moisa Clementina

## Approach:
1. Collect and clean data
2. Define a representation schema for each document type
3. Design a prompt for the Generating LLM
4. Design a prompt for the Critic LLM
5. Gather critic's results
